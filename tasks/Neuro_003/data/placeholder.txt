# Data Source: NHANES 2007-2018 (6 Combined Cycles) with Linked Mortality

This task uses publicly available data from six consecutive NHANES cycles: 2007-2008, 2009-2010, 2011-2012, 2013-2014, 2015-2016, and 2017-2018, plus linked mortality files.

## Required Data Files

### Cycle 1: NHANES 2007-2008 (suffix _E)
1. **Demographics (DEMO_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/DEMO_E.xpt
2. **Dietary Interview — Total Nutrient Intakes Day 1 (DR1TOT_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/DR1TOT_E.xpt
3. **Dietary Interview — Total Nutrient Intakes Day 2 (DR2TOT_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/DR2TOT_E.xpt
4. **Dietary Supplement Use (DSQTOT_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/DSQTOT_E.xpt
5. **Prescription Medications (RXQ_RX_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/RXQ_RX_E.xpt
6. **Blood Pressure Questionnaire (BPQ_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/BPQ_E.xpt
7. **Diabetes Questionnaire (DIQ_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/DIQ_E.xpt
8. **Standard Biochemistry Profile (BIOPRO_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/BIOPRO_E.xpt
9. **Body Measures (BMX_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/BMX_E.xpt
10. **Smoking Questionnaire (SMQ_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/SMQ_E.xpt
11. **Alcohol Use (ALQ_E.XPT)**: https://wwwn.cdc.gov/Nchs/Data/Nhanes/Public/2007/DataFiles/ALQ_E.xpt

### Cycle 2-6
Same modules with corresponding suffix:
- 2009-2010: _F suffix
- 2011-2012: _G suffix
- 2013-2014: _H suffix
- 2015-2016: _I suffix
- 2017-2018: _J suffix

### Linked Mortality Files
- **Public-Use Linked Mortality Files**: https://www.cdc.gov/nchs/data-linkage/mortality-public.htm
- Provides vital status and follow-up time through December 31, 2019
- Linked to National Death Index (NDI)

## Data Format
- Files are in SAS Transport format (.XPT)
- Can be read with R (foreign::read.xport or haven::read_xpt) or Python (pandas.read_sas)
- Merge files within cycle on SEQN (respondent sequence number)
- Append across cycles with cycle identifier
- Merge mortality data on SEQN within each cycle

## Analytic Sample
- Adults aged >= 18 years
- Complete 24-hour dietary recall data for CDAI calculation
- Complete prescription medication records for PD identification
- Eligible for mortality follow-up
- Final analytic sample: approximately 34,133 participants (510 PD, 33,623 non-PD)

## Multi-Cycle Weighting
- MEC examination weight (WTMEC2YR): divide by 6 for 12-year combined analysis
- Dietary weights (WTDR2D) when using day 2 recall data: divide by 6
- PSU and strata variables remain unchanged across cycles

## Key Analytic Notes
- CDAI = sum of 6 standardized dietary antioxidant intakes (vitamins A, C, E, carotenoids, selenium, zinc)
- Standardization: (value - mean) / SD for each component
- PD identified by prescription of PD-specific medications (carbidopa, levodopa, etc.)
- Cox regression uses person-months from MEC exam to death or end of follow-up
- RCS analysis for nonlinear dose-response characterization

## Access
- No registration or approval required
- Data freely downloadable from CDC/NCHS website
- Linked mortality files available from NCHS Data Linkage page
